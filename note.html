About

Real-Time Fires with Hunt Areas

This application displays real-time fire information from MODIS Fire Detections with the most recent Fire Emergency Closures for fire closures in Idaho from Idaho InciWeb Closures

Hunt boundaries for Controlled Hunts, Game Management Units and Elk Zones may be added by selecting a hunt from the right-hand menu. For more layers and options visit the Idaho Fish and Game Huntplanner MapCenter.

Background terrain maps, road maps and aerial imagery may be interchanged by toggling the map dropdown in the upper-left.

Data Sources

This application is constructed by "mashing up" publicly-available data sources from the U.S. Forest Service, National Interagency Fire Center FTP Server, Idaho Fish and Wildife Information System, Open Street Maps and Google Maps.

  • ActiveFireMaps WMS Tiled Layer
  • National Interagency Fire Center FTP Server
  • Idaho Fish and Wildlife Information System ArcGIS Map Server
  • Idaho Fish and Wildife Information System Open Data - KML or SHP downloads
  • MapQuest Open Street Maps Tile Service
  • Google Maps Javascript API V3

GIS Data Downloads

  • Real-Time MODIS Fire Detections Layer (kml, shp, wms)
  • Fire Closures maintained by IDFG, Boise NF and wildfire incident management teams. (ArcGIS Map Server, Google Earth Layer)

Code

Application developed by Idaho Fish and Wildlife Information System at Idaho Fish and Game. Code public domain and available on GitHub
